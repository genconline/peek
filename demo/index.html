Peek

Modern toast notifications for your web applications

Installation

Using npm:

npm install peek-notify

Using CDN:

<script src="https://unpkg.com/peek-notify/dist/peek.min.js"></script>

Basic Usage

// Initialize Peek
const peek = new Peek();

// Show a notification
peek.show({
    title: 'Hello!',
    message: 'Welcome to Peek notifications.'
});
